# Super Mario Bros. FPGA Edition

**ECE 385 Final Project - University of Illinois Urbana-Champaign - Fall 2025**

A fully-functional Super Mario Bros. clone implemented in SystemVerilog on the Urbana FPGA board. Features 2-player cooperative mode, 5 levels, 5 enemy types, hardware sprite rendering, physics engine, and SD card audio playback.
